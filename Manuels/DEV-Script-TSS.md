# Développement des scripts TSS



Ce sont les scripts qui utilisent les commandes propres à Todoist-Server.

Les commandes sont définies dans le fichier [lib/script_tss.rb](../lib/script_tss.rb).

L’aide est définie dans le fichier [Project_Scripts.md](./Project_Scripts.md), c’est dans ce fichier qu’il faut ajouter et définir toute nouvelle commande.
